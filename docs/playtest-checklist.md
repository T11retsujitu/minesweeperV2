## 固定盤面の想定ライン(参考: ネタバレ)

設計上の最短論理ライン(6ターン、被弾1回・残HP8で勝利):

1. (2,2)=1 の未開放隣接は (1,1) のみ → フラグ→起爆(敵6→4)
2. (1,2)=3 と (1,3)=1 の組合せで (0,1) が確定 → 起爆(敵4→2)
3. 同じ推理で (0,4),(1,4),(2,4) は安全 → (1,4) 開放(ここで敵攻撃、HP10→8)
4. (2,4) 開放 → (2,3)=1 から (3,4) が地雷と確定(ただし敵射程外=使う価値なし、が分かる)
5. (2,4)=1 から (1,5) 安全 → 開放すると 0 で左下が一気に開く
6. (0,4)=1 から (0,3) 確定 → 起爆(敵2→0、勝利)

深い推理を省いた場合は (0,2) vs (0,3) の五分五分ギャンブルとしても遊べる(dud なら1ターン損で情報確定)。
